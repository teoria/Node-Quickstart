# Node SinkMVC QuickStart

This repository contains a simple quickstart of how to program on a node with a good architecture using SinkMVC and good practices.

# This project was created using

* SinkMVC
* Hapi
* Sequelize
* MySQL
* Joi
* Boom
* JWT
* Gulp
* Typescript
* Docker

# Run

```
npm install
npm start
```

or

```
docker-compose up
```
